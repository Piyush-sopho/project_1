This website is a simple scrapper which scraps data from the shiksha.com url of a specific city and display the scrapped data in form of a table.
The folder contians the following files :
    connect.php : the code here connects the website to the database
    disconnect.php : used to disconnect from the database
    footer.php : simply displays the footer
    header.php : displays the header
    index.php : opening page which takes the url as input
    scrape.php : the backend script which actually scraps the data from the provided url
    result.php : displays the scrapped data in form of a table
